# Dobro-Metade-Triplo

# ⚙️ Exercício Acadêmico  – Introdução a Funções - Dobro, Metade e Triplo

Este repositório contém um **exercício simples em linguagem C**, desenvolvido com fins **educacionais**. O foco é a **introdução ao uso de funções**, prática essencial no aprendizado de programação estruturada.

---

##  📚 | Descrição do Exercício

O programa recebe um número digitado pelo usuário e utiliza **funções definidas pelo próprio programador** para realizar os seguintes cálculos:

- ✅ O **dobro** do número
- ✅ O **triplo** do número
- ✅ A **metade** do número

### 🧩 | Estrutura do Código

* Funções implementadas:

```c
float calcula_dobro(float x);
float calcula_triplo(float x);
float calcula_metade(float x);
```
---

## 💻 | Funcionamento:
  
O programa solicita um número inteiro ao usuário.

Usa as funções calcula_dobro, calcula_triplo e calcula_metade para processar os valores.

Exibe os resultados no terminal de forma clara e organizada.

---

## 🎯 | Objetivo Acadêmico

Este exercício faz parte do conteúdo de introdução à programação em C. Ele foi desenvolvido com o objetivo de:

Praticar a criação e uso de funções.

Entender melhor a modularização de código.

Desenvolver o hábito de separar responsabilidades dentro de um programa.
